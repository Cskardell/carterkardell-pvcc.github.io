Operating System Terms
Term Definiton
Operating System The software that controls the overall...
Job Que Where "jobs" waiting in mass storage wait for execution.
FIFO "FIFO" (First in First Out) is the order
in which jobs are executed in a job que.
Interactive Processing When a program being executed carries on dialouge with a user.
Time-sharing More than one user accessing a machine at once.
Multiprogramming When time is divided into intervals and then the execution of each job is
restricted to only one interval at a time.
Application Software Application software consists of the programs for performing tasks particular
to the machineâ€™s utilization.
System Software Consists of programs for peforming the tasks particular to the machine's operation.
Utility Software Consists of programs used to optimize the computer's system.
Graphical User Interface (GUI) Objects to be manipulated are depicted pictorially on the screen.
Shell Older user interface. Communicated with users through
textual messages using a keyboard and monitor screen.
Kernel Inside of an operating system; perform the very basic functions required by the computer.
Device Driver Software unit that communicates with the controllers.
File Manager Coordiantes the use of the computer's mass storage.
Memory Manager Coordinates the computer's use of main memory.
Virtual Memory The "fictional" memory created by paging.
Boot Strapping The movement of the computers OS from the main storage to memory.
ROM "Read only Memory". Permanent data stored on the memory.
Deadlock When two or more programs are blocked from progressing because each is waiting for a
resource alloted for another.
Spooling Holding data for a later, more convinient time. Avoids deadlock.
Super User (or admin.) The user who has highly privledged control over the OS.
Multi-factor Authentification Reference to something a user 'has' rather than knows. Used for unlocking items.
