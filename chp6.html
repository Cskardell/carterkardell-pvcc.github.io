Chapter 6 Glossary

Term Definition
Machine Language Sequences of instructions that are encoded as numeric digits.
Debugging Errors being located and corrected.
Assembler Converts mnemonic expressions into machine language instructions.
Assembly Language A mnemonic system for representing programs.
Translator Translates programs expressed in high-level primitives into machine-language programs.
Compiler A translation program.
Interpreter Executes the instructions as they are translated instead of recording the translated version for future use.
Cross-Platform Software Software that can be executed across multiple different device platforms.
Natural Language The languages we speak.
Formal Language Programming Languages.
Machine Independence Software that is not dependent on the properties of a particular machine.
High-Level Programming Language Closer to natural language. Abstracts programming from computer operation details.
Object-Oriented Programming Language A software system is viewed as a collection of units, called objects, each of which can perform the actions that are immediately related to itself as well as requesting actions of other objects.
Scripting Language A subset of the imperative programming languages is the collection of languages.
Data Types

  • Float- Fractional numbers
  • A sequence of charecters

    Python Collection Data Structures
  • Lists- Lists are used to store multiple items in a single variable.
  • Tuple- A collection which is ordered and unchangeable.
  • Set- A collection which is unordered, unchangeable, and unindexed.
  • Dictionary- A collection which is ordered,changeable, and does not allow duplicates.

    Function A set of instructions for performing a task that can be used as an abstract tool by other program units.
    Local Variable A variable declared within a function.
    Global Variable Variables whose scopes are not restricted to a particular part of a program.
    Source Program A program in its original, non-translated, form.
    Object Program A program into which some other program is translated by an assembler or compiler.
