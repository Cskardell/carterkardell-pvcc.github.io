Network and Internet Terms
Group Subterms
Network Classifications WAN- "Wide Area Network". Links machines over a greater distance; Perhaps
in neighboring cities or on opposite sides of the world.
---------------------------------------------------------------------------------------------------------
MAN- "Metropolitan Area Network". Links machines spanning across a local
community.
---------------------------------------------------------------------------------------------------------
LAN- "Local Area Network". Links machines in a single building or building
complex.
Protocols CSMA/CD Ethernet- "Carrier Sense, Multiple Access with Collision Detection"
This protocol dictates that each message be broadcast to all the machines on the bus.
---------------------------------------------------------------------------------------------------------
CSMA/CA Wifi- "Carrier Sense, Multiple Access with Collision Avoidance"
This protocol dictate that a machine avoids message collisions rather than detectecting
them.
Combining Networks Repeater- A device that passessignals back and forth between the two original
buses.
---------------------------------------------------------------------------------------------------------
Bridge- Like a repeater,it connects two buses, but it does not necessarily
pass all messages across the connection.
---------------------------------------------------------------------------------------------------------
Switch- Essentially a bridge with multiple connections: allowing it to
connect several buses rather than just two.
---------------------------------------------------------------------------------------------------------
Router- Special purpose computers used for forwarding messages.
---------------------------------------------------------------------------------------------------------
Gateway- The “point” at which one network is linked to an internet is
often called a gateway.
Client/Server Model P2P: Peer-to-Peer- Processes that provide service to and receive service
from each other.
---------------------------------------------------------------------------------------------------------
Distributed Systems- Consists of software units that execute as
processes on different computer.
---------------------------------------------------------------------------------------------------------
Cloud Computing- Huge pools of shared computers on the network
can be allocated for use by clients as needed.
Internet Adressing Tier 1- Very high-speed, high-capacity, international WANs.
---------------------------------------------------------------------------------------------------------
Tier 2- More regional in scope and less potent in their
capabilities.
---------------------------------------------------------------------------------------------------------
Tier 3- Provide access to the more important internet cores.
IP Adress ICANN-Where domains are registered.

DNS- "Domain Name System". Internet wide directory system
converting adresses into IP adress before transmitting the message.
---------------------------------------------------------------------------------------------------------
TLD: Top-Level Domain- The general and most widely used domain suffixes.
---------------------------------------------------------------------------------------------------------
Country Code TLDs- Top level domains for countries.
Internet Protocols FTP- Standard communication protocol used for the transfer of
computer files from a server to a client on a computer network.
---------------------------------------------------------------------------------------------------------
HTTP-"Hypertext Transfer Protocol". Where hyptertext documents
are normally transferred between browsers and webservers.
---------------------------------------------------------------------------------------------------------
SMTP- An application used by mail servers to send, receive, and
relay outgoing email between senders and receivers
World Wide Web URL- "Uniform Resource Locator". The adress given to a unique
resource on the web.
---------------------------------------------------------------------------------------------------------
Client-side activities- Anything in a web application that is
displayed or takes place on the client.
---------------------------------------------------------------------------------------------------------
Server-side activities- Anything in a web application that is
displayed or takes place on the server.
Internet Four-layer software Application Layer- Where a message originates. Provides an adress
compatible with the internets infrastructure.
---------------------------------------------------------------------------------------------------------
Transport Layer- Ensures messages are properly formatted for
transmission over the internet.
---------------------------------------------------------------------------------------------------------
Network Layer- Decides in which direction a packet should be
sent at each step along the packet’s path through the Internet.
---------------------------------------------------------------------------------------------------------
Link Layer- Transfers the packet.
Internet Is: Packet Switching Network- Grouping messages in packets that are
transmitted over the web.
---------------------------------------------------------------------------------------------------------
Fault Tolerant- The internet is able to function even if
certain components fail.
---------------------------------------------------------------------------------------------------------
Redundant- There is more than one way to route data.
Cybersecurity Malware- Malicious Software used to attack a computer system.
---------------------------------------------------------------------------------------------------------
Virus- Software that infects a computer by inserting itself
into programs that already reside in the machine.
---------------------------------------------------------------------------------------------------------
Worm- An autonomous program that transfers itself through a
network, taking up residence in computers and forwarding copies
of itself to other computers.
---------------------------------------------------------------------------------------------------------
Trojan Horse- A seemingly desirable program that is used for
malicious intentions. May lie dormit until a specific que.
---------------------------------------------------------------------------------------------------------
Spyware- Software that collects information about activities
at thecomputer on which it resides and
reports that information back to the instigator of the attack.
---------------------------------------------------------------------------------------------------------
Phishing- Asking for personal information under the guise
of authority. Often carried out via Email.
---------------------------------------------------------------------------------------------------------
DDoS Attack- Attempting to crash a server by overloading
it with information/requests.
Protection Firewall- Filtering traffic passing through a point in the
net-work
---------------------------------------------------------------------------------------------------------
Spam Filter- Filtering messages on whether or not they are spam.
---------------------------------------------------------------------------------------------------------
Proxy Server- A software unit that acts as an intermediary between a client
and a server with the goal of shielding the client from adverse actions of the server.
---------------------------------------------------------------------------------------------------------
Antivirus Software- Used to detect and remove the presence of known viruses
and other infections.
---------------------------------------------------------------------------------------------------------
Cryptography- The study of sending and receiving messages securely in the
presence of adversaries
---------------------------------------------------------------------------------------------------------
Public Key Encryption- The encryption and decryption algorithms are known,
and one of the keys is known, but the encoded message remains secure
